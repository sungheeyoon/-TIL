# 오늘 배운 것: GitHub Til 작성하기

시작일 : 2024-01-10

## 작성 규칙

- TIL 파일의 이름은 영문으로 작성하며, 공백 대신 하이픈(-)사용한다.
- 마크다운(Markdown) 사용
- 카테고리화 TIL은 카테고리에 따라 폴더를 나누어 구조화한다.
- 매일 contributions를 초록색으로 채우는 것에 집착하지 않는다.
- reference를 명시한다.
